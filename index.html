Session Storage:
The session storage property allows you to access a session storage object. Data stored in LocalStore has no expiration time setting, and data stored in sessionStore is cleared at the end of the page session except the page session is maintained while the browser is open and reloading or restoring the page still retains the original page session. Opening a page in a new tab or window will start a new session, which is different from how session cookies work
Local Storage:
The local storage property allows you to access a local storage object. Localstation equals sessionstation. The difference is that the data in the local storage has no expiration date and the data stored in the session storage is cleared at the end of the browser session (i.e. when the browser is closed).
